About this game

This game is a slightly modified game of Reversi (for history and details read this Wikipedia page).

You can play either “Classic Reversi” or “Reversi-on-Toroid”.

Rules

“Classic” rules:

  1. The BLACK player moves first. BLACK must click a square on the board, so that there exists at least one straight (horizontal, vertical, or diagonal) occupied line between the new BLACK square and another BLACK square, with one or more contiguous WHITE squares between them.
  2. After clicking the square, BLACK flips/captures (makes BLACK) all WHITE squares lying on a straight line between the new square and any old BLACK squares. Thus, a valid move is one where at least one square is reversed.
  3. Now WHITE plays. WHITE clicks an empty (GREEN) square making it WHITE, causing BLACK squares to flip.
  4. Players take alternate turns. If one player cannot make a valid move, play passes back to the other player.
  5. The game ends when neither player can legally click any of the remaining squares.

“Reversi-on-Toroid” rules:

All rules are the same as in “Classic” but the board is treated as a toroid:

  • The rightmost vertical is considered the left neighbor of the rightmost vertical.
  • The top horizontal is considered the bottom neighbor of the bottom horizontal.

How to play

Note! To go back to initial page click "Reset" button (top left corner)

To read help click "Help" button (top right corner)

How to start

To start a game with classic rules

Click "START CLASSIC REVERSI GAME" button (bottom left corner)

To start a game on toroid

Click "START TOROID REVERSI GAME" button (bottom right corner)

When a game starts

You can see (top to bottom)

  • "Reset" and "Help" buttons
  • Type of the game you are playing
  • Game board
  • Current scores for BLACK player (on left) and WHITE player (on right)
  • Message line (in the beginning it says that BLACK player must move).

To change a player's name

Click either BLACK or WHITE player's score. In the message line enter a new name for the player. Then click OK.

The names can be changed whenever you want

To choose computer ("AI") as a player

Change the player's name to "AI (level 1)" (a weaker opponent: random moves) or "AI (level 2)" (default).

Note! Only one player can be AI.

Whenever you want you can switch back from AI to human player. Just choose any "non-AI" name for the player.

To make your move

Click a green (unoccupied) square.

If your move is not valid you'll get a warning message and have one more chance to make a valid move.

If a player cannot make a valid move...

... then the opponent can move once more. (A message about that is displayed.)

When neither player can make a valid move...

... the GAME IS OVER.

To play one more game, click "Reset" and begin from the beginning.

Note! Your custom name and human/computer settings disappear after reset.
